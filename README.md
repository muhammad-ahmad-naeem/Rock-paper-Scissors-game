# ✊✋✌️ Rock Paper Scissors

A simple desktop Rock Paper Scissors game built with **Python** and **Tkinter**, styled with a warm beige theme.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-8C6A3B)

## Features

- One-click choice buttons — no typing needed
- Side-by-side reveal of your pick vs. the computer's
- Live scoreboard (Wins / Ties / Losses)
- Reset button to start fresh

## Getting Started

\`\`\`bash
git clone https://github.com/<your-username>/rock-paper-scissors.git
cd rock-paper-scissors
python rps_app.py
\`\`\`

No extra installs needed — just Python's built-in \`tkinter\`.
> Linux users may need to install it first: \`sudo apt install python3-tk\`
